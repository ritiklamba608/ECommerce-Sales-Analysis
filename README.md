# Madhav Ecommerce Sales Dashboard (Power BI)

An interactive **Power BI dashboard** that analyzes ecommerce sales performance, customer behavior, and profitability trends.  
This project provides actionable insights into revenue, profit patterns, top customers, and category performance.

---

## Project Overview
This dashboard helps stakeholders understand:
- Overall sales and profit performance  
- Monthly profit trends  
- Top contributing states and customers  
- Category-wise and sub-category profitability  
- Payment mode distribution  

It is designed for **data-driven business decision making**.

---

## Key Features
- Interactive **Quarter and Category filters**
- KPI cards for quick business summary
- Profit trend analysis by month
- Customer and state contribution insights
- Category and payment mode distribution
- Sub-category level profitability analysis

---

## Key Insights
- Total Sales: **₹438K** with **₹37K profit**
- **Clothing drives 63%** of total quantity sold
- **Maharashtra** is the top contributing state
- **COD is the most used payment mode (44%)**
- Highest profits come from **Printers and Bookcases**
- Some months show **negative profits**, indicating cost issues
